# 💩 똥 피하기 게임 (Poop Dodge Game)

하늘에서 떨어지는 똥(또는 다른 물체들)을 피하는 웹 기반 아케이드 게임입니다.  
점점 어려워지는 10단계의 스테이지에 도전하고 최고 점수를 기록해보세요!

## 🎮 게임 방법

### 조작법
*   **키보드**: `⬅️ 왼쪽 화살표` / `➡️ 오른쪽 화살표` 키를 사용하여 이동합니다.
*   **터치 (모바일)**: 화면의 `왼쪽` 또는 `오른쪽`을 터치하여 이동합니다.

### 규칙
1.  하늘에서 떨어지는 물체를 피하세요.
2.  물체에 닿으면 게임이 종료됩니다.
3.  일정 개수의 물체를 피하면 다음 스테이지로 넘어갑니다.
4.  스테이지가 올라갈수록 물체가 떨어지는 속도가 빨라지고, 더 자주 떨어집니다.
5.  총 10단계의 스테이지가 있습니다.

## ✨ 주요 기능

*   **캐릭터 선택**: 3가지 캐릭터 중 하나를 선택하여 플레이할 수 있습니다.
    *   🦈 트랄랄레로
    *   🐊 봄바르디로
    *   🥁 사후르
*   **장애물 선택**: 떨어지는 물체의 모양을 바꿀 수 있습니다.
    *   💩 똥
    *   🪨 돌
    *   👽 외계인
*   **단계별 난이도**: 10단계의 레벨 디자인으로 점진적인 난이도 상승을 경험할 수 있습니다.
*   **점수 및 랭킹**: 플레이 기록(점수, 도달 단계)이 저장되며, '명예의 전당'에서 상위 10명의 기록을 확인할 수 있습니다. (로컬 저장소 사용)
*   **반응형 디자인**: PC와 모바일 환경 모두에서 즐길 수 있습니다.

## 🛠️ 기술 스택

*   HTML5
*   CSS3
*   Vanilla JavaScript (Canvas API)

## 🚀 실행 방법

1.  이 저장소를 클론하거나 다운로드합니다.
2.  `index.html` 파일을 웹 브라우저로 엽니다.
3.  게임 시작 화면에서 캐릭터와 장애물을 선택하고 이름을 입력한 뒤 '게임 시작!' 버튼을 누릅니다.
